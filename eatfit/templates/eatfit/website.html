Técnicas Usadas
Carateristicas Elementos/técnicas Exemplos
Layouts Usou-se o CSS grid e CSS flexbox para construção de layouts Container na intro da linha 38 a 52
Responsividade Foram usadas media queries que adaptam o layout para telemovel e PC consoante o tamanho da janela do browser. Media Query na intro da linha 236 a 247
Seletores Foram usados seletores universais, de tipo, classe, id,atributo e pseudo-classe Página estilos.css da linha da linha 91 a 97(seletor de atributo)
Etiquetas HTML5 main, header, article, footer, figure, a Todas as páginas contém um footer no fim
Animações Foram feitas animações usando keyframes que permitem dar movimento e comportamentos particulares a imagens. No header da página dos exercícios, o codigo do keyframe está na pagina dos estilos.css da linha 211 a 214
Views Utilizamos várias views que são simplesmente funções Python que recebem um pedido Web e retorna uma resposta Web.
Models Utilizamos vários models que são uma fonte única e definitiva de dados. Eles contém os campos e comportamentos essenciais dos dados que nós queremos gravar.
Forms Para recolher os dados dos utilizadores utilizámos vários formulários. Para isso usamos o forms que é a biblioteca de manipulação de formulários do django.
URLS Cada expressão no módulo urls.py serve para analisar as URLs dos pedidos e invocar a view apropriada para cada padrão de URL.
JavaScript JavaScript é uma linguagem de programação de alto nível criada para ser executada em navegadores e manipular comportamentos de páginas web.
Comunicação assíncrona AJAX para comunicar com APIs Para conseguirmos apresentar a informação que queríamos na SPA, tivemos de criar este script que efetua a comunicação com a API.
